Aplicação em Angular

O objetivo da atividade é para que possamos praticar em casa e testar o que aprendemos em sala.

Assim tivemos como desafio criar quatro componentes: Home, Sobre, Cadastro e NotFound. Cada um deles possui informações diferente, nas seguintes páginas terão informações basicas sobre o aluno (acredito que eu) e a criação de um formulário para adicionar dados. Caso seja acessada uma rota inesistente também deve haver uma mensagem de erro.

Tecnologias Utilizadas

Angular: Framework para desenvolvimento de aplicações web.

Angular Material: Biblioteca de componentes para estilização e interface do usuário.
